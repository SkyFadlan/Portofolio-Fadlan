Keahlian & Kemampuan

Saya memiliki semangat dan tujuan untuk membantu bisnismu tampil menonjol di tengah persaingan.

HTML

HTML (HyperText Markup Language) adalah fondasi utama dari semua halaman web. Saya menggunakannya untuk membuat struktur konten seperti teks, gambar, dan elemen interaktif dalam sebuah website.

CSS

CSS (Cascading Style Sheets) digunakan untuk mendesain tampilan halaman web. Saya menggunakan CSS untuk membuat layout responsif, animasi, dan desain yang menarik agar website tampil profesional di berbagai perangkat.

Dart

Dart adalah bahasa pemrograman utama yang digunakan dalam Flutter. Saya menggunakan Dart untuk membangun logika aplikasi mobile, membuat tampilan dinamis, dan menghubungkan ke database serta API.

PHP

PHP adalah bahasa pemrograman server-side yang saya gunakan untuk membangun backend website dan aplikasi, terutama untuk mengelola data, autentikasi pengguna, dan menghubungkan ke database seperti MySQL.

Firebase

Firebase adalah platform backend dari Google yang saya gunakan untuk autentikasi pengguna, database real-time, dan penyimpanan cloud.

Laravel

Laravel adalah framework PHP modern yang saya gunakan untuk membangun aplikasi web yang cepat, aman, dan terstruktur. Dengan Laravel, saya membuat autentikasi, dan fitur manajemen data.

Flutter

Flutter adalah framework UI dari Google untuk membuat aplikasi mobile secara cross-platform (Android & iOS) dari satu basis kode. Saya menggunakannya untuk membuat aplikasi dengan antarmuka modern dan interaktif.

Figma

Figma adalah alat desain UI/UX berbasis cloud yang saya gunakan untuk membuat prototype dan wireframe aplikasi maupun website sebelum dikembangkan. Ini memudahkan kolaborasi dan visualisasi desain.

Canva

Canva adalah alat desain grafis berbasis cloud yang saya gunakan untuk membuat prototype, wireframe, serta elemen visual aplikasi maupun website secara cepat dan praktis.
